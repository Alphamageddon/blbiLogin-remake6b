# 波比登录 [blbiLogin]

##### 波比登录 [blbiLogin] 是基于高版本 Minecraft Paper 核心制作的玩家登录解决方案

| 项       | 值                            |
| -------- | ----------------------------- |
| 插件名   | 波比登录 [blbiLogin]          |
| 最新版本 | Alpha 1.0                     |
| 最低要求 | Minecraft ≥ 1.21 && Java ≥ 21 |

### 基本命令

| 命令              | 缩写        | 作用                     | 权限                       |
| ----------------- | ----------- | ------------------------ | -------------------------- |
| /register <密码>  | /reg <密码> | 玩家注册                 |                            |
| /login <密码>     | /l <密码>   | 玩家登录                 |                            |
| /blbilogin <参数> | none        | 插件主命令<br /><reload> | none<br />blbilogin.reload |
